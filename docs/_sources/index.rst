**Le système d'exploitation pour Robot (ROS) est un ensemble de bibliothèques logicielles et d'outils pour construire des applications robotiques.**
Depuis les pilotes matériel, des algorithmes de l'état de l'art jusqu'à de puissants logiciels de développement, ROS a les
outils libres et ouverts dont vous avez besoin pour votre prochain projet robotique.

Depuis que ROS a démarré en 2007, beaucoup de choses ont changé en robotique et dans la communauté ROS.
Le but du projet ROS 2 est de s'adapter à ces changements, en s'appuyant sur les aspects positifs de ROS 1 et d'améliorer les aspects négatifs.

Ce site contient la documentation française de ROS 2.
Si vous cherchez la documentation ROS-1, il faut aller sur le `wiki ROS <https://wiki.ros.org>`__.

Si vous utilisez ROS 2 dans votre travail, vous pouvez aller voir :doc:`Citations <Citations>` pour citer ROS 2.

Autres ressources ROS
---------------------

* `ROS Answers <https://answers.ros.org/questions/>`__ (ROS 1, ROS 2)

  - Site web (en anglais) de questions et réponses, similaire à `Stack Exchange <https://stackexchange.com/>`_
  - Voir :ref:`Contact Page <Using ROS Answers>` pour plus d'informations

* `ROS Enhancement Proposals (REPs) <https://ros.org/reps/rep-0000.html>`__ (ROS 1, ROS 2)

  - Propositions pour de nouveaux concepts et de nouvelles conventions

* `ROS Discourse <https://discourse.ros.org/>`__ (ROS 1, ROS 2)

  - Forum pour des discussions générales et des annonces à destination de la communauté ROS.
  - Voir :ref:`Contact Page <Using ROS Discourse>` pour plus d'informations

* `ROS Index <https://index.ros.org/>`__ (ROS 1, ROS 2)

  - Une liste indexée de tous les paquets (i.e. `Python Package Index (PyPI) <https://pypi.org/>`_ pour les paquets ROS)
  - Voir quelles version de ROS un paquet supporte
  - Lien vers le dépôt d'un paquet, la documentation de l'interface, ou le site web
  - Inspecter la license d'un paquet, de son type de construction, les maintaineurs, le statut, et les dépendences
  - Obtenir plus d'informations sur un paquet `ROS Answers <https://answers.ros.org/questions/>`__

* `ROS Prerelease <http://prerelease.ros.org/>`__ (ROS 1)

  - Générer des commandes pour simuler la `ferme de compilation ROS <https://build.ros.org/>`_ sur votre machine
  - Ne montre ques les distributions ROS 1

* `ROS Robots <https://robots.ros.org/>`__ (ROS 1, ROS 2)

  - Mets en avant les projets robotiques de la communauté
  - Instructions pour ajouter un robot

* `ROS Wiki <http://wiki.ros.org/>`__ (ROS 1)

  - La documentation ROS 1 et du contenu modifiable par les utilisateurs
  - Actif au moins jusqu'à la fin de vie de la dernière distribution ROS 1

* `ROS.org <https://www.ros.org/>`__ (ROS 1, ROS 2)

  - Il s'agit de la page principale de ROS 1 et ROS 2 , avec une description haut niveau de ROS et des liens vers les autres sites ROS
