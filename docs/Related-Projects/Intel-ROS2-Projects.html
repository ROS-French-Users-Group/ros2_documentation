Projets Intel ROS 2

Projet Intel® Robotics Open Source (Intel® ROS Project) pour permettre la détection/localisation/suivi d’objets, la détection de personnes, la détection de véhicules, l’analyse des points de préhension du bras de robot de l’industrie avec des types de technologies et de plates-formes Intel, y compris CPU, GPU, Intel® Movidius ™ NCS backend d’apprentissage en profondeur optimisé, FPGA, `Intel® RealSense™ < https://www.intel.com/content/www/us/en/architecture-and-technology/realsense-overview.html>`__ caméra, etc.

Projets clés

Nous travaillons sur les projets ROS 2 ci-dessous et publions progressivement le code source via https://github.com/intel/ ou ROS 2 GitHub repo.

  • ROS2 OpenVINO : package ROS 2 pour Intel® Visual Inference et Neural Network Optimization Toolkit pour développer des solutions de vision par ordinateur multiplateformes.

  • Caméra ROS2 RealSense : package ROS 2 pour les caméras série Intel® RealSense™ D400

  • ROS2 Movidius NCS : package ROS 2 pour la détection d’objets avec Intel® Movidius™ Neural Computing Stick (NCS).

  • Messages d’objet ROS2 : messages ROS 2 pour l’objet.

  • ROS2 Object Analytics : package ROS 2 pour la détection, le suivi et la localisation d’objets 2D/3D.

  • Filtres de messages ROS2 : package ROS 2 pour la synchronisation des messages avec horodatage.

  • Pont CV ROS2 : package ROS 2 pour faire le pont avec openCV.

  • ROS2 Object Map : package ROS 2 pour marquer la balise des objets sur la carte en cas de SLAM sur la base des informations fournies par l’analyse d’objets ROS 2.

  • ROS2 Moving Object : package ROS 2 pour fournir des informations sur le mouvement de l’objet (comme la vitesse de l’objet sur l’axe x, y, z) sur la base des informations fournies par l’analyse d’objet ROS 2.

  • Bibliothèque ROS2 Grasp : package ROS 2 pour l’analyse de la position de préhension et compatible avec `MoveIt <https://github.com/ros-planning/moveit2.git> `__ saisir les interfaces.

  • Navigation ROS2 : package ROS 2 pour la navigation robotique, il est déjà intégré à la version ROS 2 Crystal.

  • Intel Robot DevKit (SDK) : un projet open source qui permet aux développeurs de créer, personnaliser, optimiser et déployer facilement et efficacement une pile logicielle de robot sur un mobile autonome Plateforme Robot (AMR) basée sur le framework Robot Operating System 2 (ROS 2).

Référence

Les composants ROS sur : https://wiki.ros.org/IntelROSProject montrent la relation entre ces packages, qui s’applique également à ROS 2.
